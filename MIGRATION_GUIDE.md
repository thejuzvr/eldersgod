## Требуемые изменения в БД

### Вариант 1: Автоматическая миграция (рекомендуется)

```bash
npm run db:push
```

Эта команда применит все изменения схемы автоматически.

### Вариант 2: Ручная миграция

Выполните SQL из файла `migrations/add_new_features.sql`:

```bash
psql -U your_user -d your_database -f migrations/add_new_features.sql
```

### Что будет добавлено:

1. **Новое поле в heroes:**
   - `current_location` - текущая локация героя

2. **Новая таблица hero_quests:**
   - Задания для каждого героя
   - Подзадания и прогресс
   - Награды за выполнение

3. **Новая таблица hero_skills:**
   - Умения героев
   - Уровни умений
   - Категории (combat, magic, stealth, social, absurd)

4. **Новая таблица user_ideas:**
   - Идеи от игроков
   - Категории и статусы

## После миграции

1. Перезапустите сервер:
```bash
npm run dev
```

2. Проверьте новые страницы:
   - `/game` - обновленный Dashboard
   - `/game/hero` - детальная страница героя
   - `/game/inventory` - инвентарь
   - `/game/skills` - умения
   - `/game/map` - карта мира
   - `/game/arena` - улучшенная арена
   - `/game/leaderboard` - пантеоны
   - `/game/wiki` - энциклопедия
   - `/game/ideas` - ящик идей

3. Новые функции:
   - Божественный пульт на главной странице
   - Мини-игры при клике на мысли героя
   - Квесты с прогресс-барами
   - Навигационное меню сверху

## Заметки

- События теперь генерируются каждые 15-180 секунд (рандомно)
- Герои получают урон от событий
- При смерти герой воскрешается с 30% HP и теряет 10% золота
- Новые события отображаются сверху в дневнике
- Все цвета переведены на строгую светлую палитру
