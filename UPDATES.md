# 🎉 Обновления проекта Elder Scrolls: Автоприключения

## 🎨 Глобальное обновление UI

### Новая цветовая схема
- **Светлая строгая палитра** без ярких акцентов
- Белый фон (#FFFFFF), светло-серые акценты
- Минималистичный дизайн в стиле современных приложений
- Шрифт Inter для лучшей читаемости

## 🎮 Игровые механики

### Обновления системы событий
- ✅ **GameTick увеличен:** 15-180 секунд с рандомизацией
- ✅ **Типизация событий:** 9 категорий (in_combat, exploration, social, quest, trade, rest, danger, reward, event)
- ✅ **Дневник реверсирован:** новые события сверху
- ✅ **Система урона HP:** герои получают урон, есть механика смерти
- ✅ **Локации:** герои перемещаются по Тамриэлю

### Система смерти
- При HP ≤ 0 герой воскрешается с 30% HP
- Штраф: -10% золота
- Полное восстановление при повышении уровня

## 🎯 Новые функции

### 1. 🎮 Мини-игры (4 штуки)
- **Угадай мысль дракона** - выбор из вариантов
- **Угадай число** - от 1 до 10, 3 попытки
- **Быстрая математика** - решить пример за 15 секунд
- **Память** - найти пары за 15 попыток

### 2. ⚡ Божественный пульт
- **Благословение:** +20 HP, +50 опыта, +25 золота
- **Наказание:** -15 HP, +30 опыта
- Кулдаун: 60 секунд
- Стоимость: 30% силы молитвы

### 3. 📜 Квесты с подзаданиями
- Привязаны к героям (не к пользователям)
- Прогресс-бары для отслеживания
- Чекбоксы для подзаданий
- Награды за выполнение

### 4. 📚 Система умений
- 28+ абсурдных умений
- 5 категорий: боевые, магия, скрытность, социальные, абсурдные
- Система уровней умений
- Случайное получение во время событий

### 5. 🗺️ Карта мира
- Отображение текущей локации героя
- Список всех регионов и городов
- Автоматические перемещения

### 6. ⚔️ Улучшенная арена
- Компонент боя с прогресс-баром
- 18 забавных реплик во время боя
- Пошаговая система (10 раундов по 1.5 сек)
- Учет характеристик героев
- Фильтры боев (все/победы/поражения)
- Развертываемые детали каждого боя

### 7. 🏆 Пантеоны (Рейтинги)
- 7 различных рейтингов
- Мастерства (по уровню)
- Строительства (по золоту)
- Живучести (по HP)
- Созидания (по душам драконов)
- И другие...

### 8. 🎒 Инвентарь
- Отдельная страница с фильтрами
- Система редкости предметов
- Экипировка/снятие предметов
- Бонусы от снаряжения

### 9. 👤 Страница героя
- Детальная информация
- Слоты снаряжения
- Список умений
- Суммарные бонусы

### 10. 📖 Wiki страница
- 6 разделов с информацией
- О игре, механиках, локациях
- Существа и предметы
- Как помочь проекту

### 11. 💡 Ящик для идей
- Форма обратной связи
- 13 категорий для предложений
- Сохранение в БД

### 12. 🧭 Навигационное меню
- Быстрый доступ ко всем страницам
- Адаптивный дизайн
- Подсветка активной страницы

## 🗄️ Изменения в БД

### Новые таблицы
1. `hero_quests` - задания героев
2. `hero_skills` - умения героев
3. `user_ideas` - идеи игроков

### Новые поля
- `heroes.current_location` - текущая локация

### Миграция

**Вариант 1 (рекомендуется):**
```bash
npm run db:push
```

**Вариант 2:**
```bash
psql -U your_user -d your_database -f migrations/add_new_features.sql
```

## 🔧 API Endpoints

### Новые endpoints
- `POST /api/hero/[heroId]/divine-action` - божественные действия
- `POST /api/items/[itemId]/equip` - экипировать предмет
- `POST /api/items/[itemId]/unequip` - снять предмет
- `POST /api/skills/random` - дать случайное умение
- `POST /api/ideas/submit` - отправить идею

## 🚀 Запуск проекта

1. Примените миграции БД:
```bash
npm run db:push
```

2. Запустите сервер:
```bash
npm run dev
```

3. Откройте браузер:
```
http://localhost:5173
```
