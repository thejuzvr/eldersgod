# 🚀 Quick Start Guide

## Быстрый старт проекта

### 1. Применить изменения в БД

```bash
npm run db:push
```

Эта команда автоматически синхронизирует схему БД с вашими изменениями.

### 2. Запустить сервер разработки

```bash
npm run dev
```

### 3. Открыть браузер

```
http://localhost:5173
```

## ⚠️ Важно!

Если команда `db:push` не сработала, выполните миграцию вручную:

```bash
# Для PostgreSQL
psql -U your_user -d your_database -f migrations/add_new_features.sql
```

## 📋 Новые возможности

После обновления доступны:

1. **Навигационное меню** - сверху страницы
2. **Божественный пульт** - справа на главной странице
3. **Мини-игры** - клик на мысли героя
4. **Квесты** - справа на главной странице
5. **Карта мира** - `/game/map`
6. **Умения** - `/game/skills`
7. **Инвентарь** - `/game/inventory`
8. **Герой** - `/game/hero`
9. **Wiki** - `/game/wiki`
10. **Ящик идей** - `/game/ideas`
11. **Пантеоны** - `/game/leaderboard`

## 🎮 Геймплей

- События генерируются каждые **15-180 секунд** (рандомно)
- Герои **получают урон** от событий
- При смерти **воскрешение** с 30% HP
- **Новые события сверху** в дневнике
- Можно **влиять на героя** через божественный пульт
