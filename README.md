# Elder Scrolls: Автоприключения ⚔️

Авто-RPG игра в мире The Elder Scrolls с абсурдным юмором в стиле Godville. Ваш герой автоматически переживает невероятные приключения в мире, где драконы пекут кексы, носки телепортируются, а грибы становятся королями.

## 🎮 Особенности

- **Автоматические события**: Герой живёт своей жизнью, генерируя события каждые 10-30 секунд
- **50+ абсурдных событий**: Битвы с кексами-драконами, философские беседы с лошадьми, дождь из сыров
- **40+ уникальных предметов**: Носки-телепорты, медовые шлемы, сахарные бомбы и многое другое
- **100+ мыслей героя**: Абсурдные размышления вашего персонажа
- **Арена PvP**: Сражайтесь с героями других игроков с абсурдными комментариями
- **Встречи героев**: Случайные взаимодействия между героями разных игроков в зависимости от локации
- **Система Умений**: Находите Книги Навыков и улучшайте характеристики вашего героя
- **Умная экипировка**: Герой автоматически оценивает найденные предметы и надевает лучшие
- **Мини-игры**: Участвуйте в викторине "Шепот предков" (нажмите на мысли героя) и получайте награды
- **Система Уведомлений**: Встроенная система оповещений (toasts + выпадающий список) держит вас в курсе важных событий
- **Божественное Вмешательство**: Наказывайте или благословляйте героя с помощью пульта
- **Улучшенный ИИ Героя**: Поведение героя теперь логично зависит от его состояния (поиск таверны при низком здоровье, битвы во время квестов)

## 🛠 Технологии

- **Frontend**: SvelteKit + TypeScript
- **Styling**: Tailwind CSS (кастомная vintage-фэнтези тема)
- **Database**: PostgreSQL + DrizzleORM
- **Cache**: Redis (для очередей событий и кэширования)
- **Authentication**: JWT + bcrypt

## 📦 Установка

### Требования

- Node.js 18+
- PostgreSQL 14+
- Redis 7+

### Шаги установки

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd eldersgod
```

2. Установите зависимости:
```bash
npm install
```

3. Создайте файл `.env` на основе `.env.example`:
```bash
cp .env.example .env
```

4. Отредактируйте `.env` и укажите свои настройки:
```env
DATABASE_URL=postgresql://postgres:postgres@localhost:5432/eldersgod
REDIS_URL=redis://localhost:6379
JWT_SECRET=your-super-secret-jwt-key
PUBLIC_APP_URL=http://localhost:5173
```

5. Создайте базу данных:
```bash
createdb eldersgod
```

6. Сгенерируйте и примените миграции:
```bash
npm run db:generate
npm run db:push
```

7. Заполните базу начальными данными:
```bash
npm run db:seed
```

8. Запустите сервер разработки:
```bash
npm run dev
```

Приложение будет доступно по адресу: http://localhost:5173

## 🐳 Docker (рекомендуется)

Используйте Docker Compose для простого запуска всех сервисов:

```bash
docker-compose up -d
```

Это запустит:
- PostgreSQL на порту 5432
- Redis на порту 6379
- Приложение на порту 5173

## 🔧 NPM Scripts

- `npm run dev` - Запуск dev сервера
- `npm run build` - Сборка для production
- `npm run preview` - Предпросмотр production сборки
- `npm run check` - Проверка типов TypeScript (svelte-check)
- `npm run db:generate` - Генерация миграций
- `npm run db:push` - Применение миграций
- `npm run db:studio` - Открыть Drizzle Studio
- `npm run db:seed` - Заполнение БД начальными данными

## 🌍 Production Deployment

1. Убедитесь, что все env переменные установлены
2. Соберите приложение: `npm run build`
3. Запустите production сервер: `node build`

Рекомендуется использовать:
- PM2 для управления процессом
- Nginx как reverse proxy
- PostgreSQL и Redis на отдельных серверах
